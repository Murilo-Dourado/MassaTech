Protótipo de Baixa Fidelidade

Esta pasta contém as imagens e evidências do protótipo de baixa fidelidade desenvolvidas para o Projeto Integrador I – MassaTech.

O protótipo de baixa fidelidade foi elaborado com o objetivo de representar, de forma inicial, a estrutura das telas e o fluxo de navegação do sistema proposto para a empresa parceira Parada da Pizza. As imagens aqui armazenadas demonstram a disposição dos elementos da interface e auxiliam na validação dos requisitos definidos pela equipe antes do desenvolvimento do sistema.

Os arquivos presentes nesta pasta servem como documentação do processo de planejamento e concepção da solução, permitindo visualizar a evolução do projeto e as decisões tomadas durante a fase de prototipação.
